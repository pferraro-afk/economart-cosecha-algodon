# Control de Cosecha de Algodón — Economart

Sistema para automatizar el seguimiento de cosecha de algodón usando datos de la API Finnegans (remitos de algodón Duhau).

## Qué hace

- Fetch de datos desde la API Finnegans (Análisis Remitos Algodón Duhau)
- Procesamiento de remitos por lote/establecimiento
- Informes dinámicos de avance de cosecha

## Stack

- Python 3.11+
- pandas, requests
- Claude Code para desarrollo y análisis
